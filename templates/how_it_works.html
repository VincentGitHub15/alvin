How it Works

Backend

MedicalMap.us is hosted with Digital Ocean, a cloud infrustructure provider. We are using Ubuntu, a Linux based operating system, and set it up with Apache, the web server. This means our application is ready to scale.

We used PostgreSQL for our database with the intention of using it to store images in the future. It is an object relational database that uses the standard Structured Query Language, SQL.

We used the 'lightweight Python web framework' Flask to route our pages, and as such, our controller is written in Python.

All this constitutes a LAPP stack: Linux, Apache, PostgreSQL, and Python.

Additionally, we used the Google Maps API to display the map as it is the most familiar for users.

Frontend

As this project was created for Alvin Healthcare, the headers and footers are theirs, and we did little to design those. However we used HTML, CSS, and Javascript to display and format the map, as well as the markers on the map. We also used Javascript to make AJAX resquests to update the map, and to manipulate returned JSON objects.
